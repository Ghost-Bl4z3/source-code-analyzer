# Source Code Analyzer 🔍

A simple python tool for analyzing web source code and extracting useful recon data

## 🚀 Features

- Extracts URLS
- Extracts Javascript Files
- Extracts Ip Addresses
- Extracts Emails
- Extracts interesting endpoints using:
  -Custom wordlist
  -internal wordlist


# ☠️ Usage


### Analyze a file
- python Source_Code_Analyzer.py --file sample-source.html


## Use Internal Wordlist
- python Source_Code_Analyzer.py --file sample-source.html --internal


## Use Custom Wordlist
- python Source_Code_Analyzer.py --file sample-source.html --wordlist wordlist.txt
